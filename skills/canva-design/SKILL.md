## O que faz
- Cria designs profissionais no Canva a partir de descrições textuais
- Gera apresentações completas com slides estruturados
- Cria posts para redes sociais (Instagram, LinkedIn, Twitter, Facebook)
- Redimensiona designs para diferentes formatos e plataformas
- Aplica identidade visual consistente (cores, fontes, logos)
- Cria banners, thumbnails, capas e mockups
- Sugere layouts e templates baseados no conteúdo

## Quando usar
Use ao precisar de assets visuais rápidos, apresentações, posts para redes sociais, ou qualquer design gráfico. Ideal para criar branding consistente sem sair do terminal.

## Como usar
Descreva o que precisa: formato, tema, cores, texto. Exemplo: "Use canva-design para criar um post Instagram 1080x1080 promovendo o lançamento do produto X."
