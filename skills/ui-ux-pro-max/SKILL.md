## O que faz
- Sugere paletas de cores harmoniosas baseadas em teoria das cores
- Recomenda pares de fontes tipográficas profissionais (headings + body)
- Aplica princípios de design: hierarquia visual, espaçamento, alinhamento, contraste
- Gera sistemas de design completos (cores, tipografia, spacing, borders, shadows)
- Segue diretrizes de acessibilidade WCAG (contraste, legibilidade, foco)
- Cria layouts responsivos para mobile, tablet e desktop
- Aplica padrões de UI: cards, modais, navegação, formulários, tabelas
- Fornece HTML/CSS/Tailwind components estilizados

## Quando usar
Use ao criar interfaces do zero, refatorar UIs existentes, ou quando precisar de decisões de design rápidas mas profissionais.

## Como usar
Descreva o tipo de interface e o stack (React, Tailwind, etc). Exemplo: "Use ui-ux-pro-max para criar um design system para um dashboard financeiro com Tailwind CSS."
