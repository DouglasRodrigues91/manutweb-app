## O que faz
- Ativa agentes especializados para cada fase do desenvolvimento:
  - **analyst**: Análise de requisitos e levantamento
  - **architect**: Definição de arquitetura e decisões técnicas
  - **dev**: Implementação de código
  - **qa**: Testes e validação de qualidade
  - **devops**: Infraestrutura e deploy
  - **ux-expert**: Design de interface e experiência
- Orquestra squads de agentes para tarefas complexas
- Mantém contexto compartilhado entre agentes via documentos estruturados
- Gera PRD (Product Requirements Document) e documentos de arquitetura
- Coordena desenvolvimento contexto-consciente

## Quando usar
Use em projetos full-stack complexos que exigem múltiplos papéis, ou quando precisar de uma abordagem estruturada com planejamento antes da execução.

## Como usar
Descreva o projeto e o que precisa ser feito. A skill ativa os agentes apropriados automaticamente baseado na fase do desenvolvimento.
